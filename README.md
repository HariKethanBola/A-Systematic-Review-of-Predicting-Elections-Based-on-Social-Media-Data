# 📊 A Systematic Review of Predicting Elections Based on Social Media Data

## 📌 Overview

This project presents a systematic review of predicting election outcomes using social media data. It analyzes public opinion by applying machine learning and natural language processing (NLP) techniques on user-generated content from social platforms.

---

## 🎯 Objective

The main objective of this project is to evaluate different machine learning approaches for election prediction and identify the most effective model based on performance.

---

## 🤖 Machine Learning Models Used

* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* Naive Bayes
* Logistic Regression
* Random Forest

---

## 🏆 Results

Among all the models, **Support Vector Machine (SVM)** achieved the highest accuracy and performed better on the dataset. Hence, it was selected as the final model for prediction.

---

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/HariKethanBola/A-Systematic-Review-of-Predicting-Elections-Based-on-Social-Media-Data.git
```

2. Navigate to the project folder

```bash
cd A-Systematic-Review-of-Predicting-Elections-Based-on-Social-Media-Data
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the project

```bash
python main.py
```
